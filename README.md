# 📮 Consulta de CEP — PyQt5

Aplicação desktop em Python com interface gráfica que busca endereços brasileiros em tempo real pelo CEP, integrando a API pública ViaCEP.

---

## 📋 Funcionalidades

| Funcionalidade | Descrição |
|---|---|
| Busca de CEP | Consulta endereço em tempo real via API ViaCEP |
| Preenchimento automático | Logradouro, bairro, cidade e UF preenchidos automaticamente |
| Validação de entrada | Verifica campo vazio, CEP inválido e erros de conexão |
| Limpar campos | Reseta todos os campos para nova consulta |

---

## 🛠️ Tecnologias

- Linguagem: **Python 3**
- Interface: **PyQt5**
- API: **ViaCEP** — sem necessidade de autenticação
- Dependências externas: apenas `requests` e `PyQt5`
